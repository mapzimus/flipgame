# Parrott Flip

Pirate parrots. Bottle-flip physics. Eye patches on every bird.

**Play:** https://mapzimus.github.io/flipgame/parrott-flip/

Pick a colored parrot (Captain Squawk, Pegleg Polly, Doubloon Dave, …). Same Matter.js flick physics as Flip Game — just painted as a standing pirate parrot.

Menu stays normal: Easy / Medium / Hard, Forgiving / Standard / Pro.

No build step — open `index.html` or serve the folder.
